## 🖥️ Projeto

Esse é um projeto Web Responsivo de curriculum portfolio para ser usado por desenvolvedores Jr.

## 🚀 Tecnologias

Esse projeto foi desenvolvido por mim usando como principais tecnologias:

- Astro
- HTML
- Tailwind CSS
- Git e Github
- Vercel
